# 작업 로그

이 프로젝트의 ingest, query, lint, init 등 위키 운영 이벤트를 시간순으로 누적하는 append-only 로그다. 새 항목은 파일 **끝에 추가** 한다 (역순 아님).

## 항목 포맷

```
## [YYYY-MM-DD] {ingest|query|lint|init|other} | 짧은 제목
- 한 줄 본문 (영향 받은 페이지, 결정 사항, 후속 작업 등)
- 필요하면 1~3줄 추가
```

- 헤더 prefix를 일정하게 유지하면 `grep "^## \[" system/LOG.md | tail -5` 로 최근 이력을 빠르게 볼 수 있다.
- 본문은 사실 위주로 짧게. 자세한 분석은 `01_analysis/`에 두고 링크한다.
- 같은 날 여러 항목이 있으면 발생한 순서대로 추가한다.
